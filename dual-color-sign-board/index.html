3D print - Dual color Caution Sign Board

Posted by Rakshith BK 2 years ago

1 min read

3d

This was my first attempt at printing with multiple filaments on the same object. My printer is a single extruder system and each color/filament that you use, the print has to be paused and the filament has to be replaced manually.
And Each time the filament is changes, the hot end has to be primed and then the print should be resumed.

I made this to be kept alongside my cactus planter, on my office desk. This is the planter, if you are curious.
